Tree Searches in R

2017-09-04

Motivation

R is often derided for not having ‘traditional’ computer science data structure support or common implementations of well-known algorithms. In this post, we implement breadth-first search using two (relatively) new packages, rstackdeque and data.tree.

Sample data

We use a small dataset of 2014 GNI data Here, we have a set of countries, with their ISO 3 letter country code, population and their GNI number. The tree structure is based on which continent the countries belong to.
